How to use

Send SVG file, then you can get embroidery machine format.

Compile svg2emb by yourself

svg2emb is licensed under BSD 3-Clause License. You can download source code at: https://github.com/hanabu/svg2emb

On UNIX like system (*BSD, Linux, MacOS), you can compile svg2emb as following instruction. 

$ git clone https://github.com/hanabu/svg2emb.git
$ mkdir svg2emb/build
$ cd svg2emb/build
$ cmake ..
$ make

Then you can get svg2emb on svg2emb/build directory.
